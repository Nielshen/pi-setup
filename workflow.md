Workflow (ab sofort)                                                                                                                                                                                                                                        
                                                                                                                                                                                                                                                                        
 Statt manuellem cp + git commit:                                                                                                                                                                                                                                       
                                                                                                                                                                                                                                                                        
 1. Pi live anpassen (in ~/.pi/agent/... oder via meiner Tools).                                                                                                                                                                                                        
 2. In einem Terminal mit aktivem PATH-Eintrag:                                                                                                                                                                                                                         
    ```bash                                                                                                                                                                                                                                                             
      pi-setup-sync                  # kopiert alle Extensions/Themes/Skills + settings + mcp, validiert, committet, pusht                                                                                                                                              
      pi-setup-sync --dry-run        # nur vorschauen                                                                                                                                                                                                                   
      pi-setup-sync "Commit-Message" # eigene Nachricht                                                                                                                                                                                                                 
      pi-setup-sync --no-push        # nur lokal committen                                                                                                                                                                                                              
    ```
